============
Installation
============


Stable release
--------------

To install toy_python_package, run this command in your terminal:

.. code-block:: console

    $ pip install -u toy_python_package

This is the preferred method to install toy_python_package, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for toy_python_package can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/heidi-ye/toy_python_package

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/heidi-ye/toy_python_package/tarball/main

Once you have a copy of the source, you can install it. The method of installation will depend on the packaging library being used.

For example, if `setuptools` is being used (a setup.py file is present), install toy_python_package with:

.. code-block:: console

    $ python setup.py install

If `poetry` is being used (poetry.lock and pyproject.toml files are present), install toy_python_package with:

.. code-block:: console

    $ poetry install


.. _Github repo: https://github.com/heidi-ye/toy_python_package
.. _tarball: https://github.com/heidi-ye/toy_python_package/tarball/master
